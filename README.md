## 📌 Description

**Malware Hunter** est un **jeu sérieux mobile 3D** développé sur Android qui place le joueur dans la peau d'un **analyste SOC** chargé de protéger un système informatique contre des attaques de logiciels malveillants animés par une **intelligence artificielle NavMesh**.

Le jeu transforme des concepts abstraits de cybersécurité en **expériences ludiques et mémorables**, accessibles à tous sans formation technique préalable. À travers deux niveaux immersifs et un module de réalité augmentée, le joueur apprend à identifier et neutraliser les trois grandes familles de malwares : **Virus, Spyware et Ransomware**.

---

## 🎯 Idée & Objectifs

> *"Et si la meilleure façon d'apprendre la cybersécurité, c'était de jouer contre les malwares ?"*

Le nombre de malwares a explosé de **17 millions en 2008** à **plus d'1 milliard** aujourd'hui. Pourtant, aucun jeu mobile grand public ne traite cette menace de façon pédagogique. **Malware Hunter** comble ce vide.

| &nbsp; | Objectif |
|--------|----------|
| 🎮 | Développer un jeu mobile 3D éducatif et engageant sur Android |
| 🧠 | Sensibiliser aux menaces : Virus · Spyware · Ransomware |
| 🤖 | Implémenter une IA NavMesh pour les comportements des malwares |
| 🔬 | Intégrer un module AR pour explorer les composants PC |
| 🔐 | Authentification complète via Firebase |
| 📚 | Enseigner les bonnes pratiques de cybersécurité par le gameplay |
| 📱 | Livrer une application Android complète et fonctionnelle |

## 🚀 Comment utiliser ce projet

### ✅ Prérequis

- [ ] 🎮 [Unity Hub](https://unity.com/download) + **Unity 2022.3 LTS** + module **Android Build Support**
- [ ] 🔧 [Git](https://git-scm.com/download/win)
- [ ] 📦 [Git LFS](https://git-lfs.com) ← **obligatoire**
- [ ] 📱 Appareil Android compatible **Google ARCore**

---

### Étape 1 — Initialiser Git LFS

```bash
git lfs install
```

### Étape 2 — Cloner le repository

```bash
git clone https://github.com/MounaTaleb/Mouna--Taleb-L3DSIG2-2025-2026-.git
cd Mouna--Taleb-L3DSIG2-2025-2026-
```

> ⏳ Le clone peut prendre plusieurs minutes — les assets 3D sont gérés par Git LFS.

### Étape 3 — Ouvrir dans Unity Hub

```
Unity Hub → Add → Add project from disk
→ Sélectionner le dossier "Viro Guard"
→ Sélectionner Unity 2022.3 LTS → Open
```

> ⚠️ La première ouverture régénère `Library/` automatiquement — prévoir **5 à 15 minutes**.

### Étape 4 — Configurer Firebase

```
Assets → Firebase → Setup Wizard
→ Vérifier que google-services.json est présent dans Assets/
→ Attendre la résolution automatique des dépendances Android
```

### Étape 5 — Lancer en éditeur

```
File → Build Settings → Android → Switch Platform
▶ Play pour tester en éditeur
```

### Étape 6 — Build APK Android

```
File → Build Settings → Build
→ Choisir un dossier de sortie
→ Installer l'APK sur votre appareil Android
```
